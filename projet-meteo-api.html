Détails du Projet

Ce dashboard illustre ma capacité à récupérer des données externes, gérer la géolocalisation, prévoir des fallbacks robustes et transformer le tout en une interface de supervision claire.

Ce que ce projet démontre

Consommation d'API, gestion des erreurs, géolocalisation navigateur, journalisation des états et restitution lisible d'une donnée temps réel.

Technologies

JavaScript Fetch API Geolocation API Open-Meteo UI Temps Réel

Application réelle

La logique de cette page se retrouve dans les dashboards de télémétrie, les interfaces de supervision industrielle et les salles de contrôle : agréger de la donnée, gérer l'incertitude et présenter l'information utile sans bruit.
